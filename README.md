# 中国状态分析｜多源数据监测框架

> 本文档用于后续系统性调研中国当前经济、金融、社会、资本流动与政策状态。目标不是用单一数据证明某个结论，而是建立一套可持续迭代、可交叉验证、可追踪变化的“国家状态监测框架”。

---

## 0. 文档定位

本框架基于历史上各国经济大萧条、金融危机、银行危机、货币大贬值、恶性通胀、主权债务危机、房地产泡沫破裂和资产负债表衰退案例，对中国当前状态进行多源数据监测。

重点借鉴的历史样本包括：

- 美国 1929–1933 大萧条：银行危机、信贷收缩、失业与需求崩塌。
- 德国魏玛 1921–1923：财政失控、货币信用崩溃、恶性通胀。
- 日本 1990s：房地产泡沫破裂、资产负债表衰退、长期通缩与低增长。
- 北欧 1990s：房地产与银行坏账危机。
- 墨西哥 1994：短期美元债、资本外逃、比索危机。
- 亚洲金融危机 1997：固定汇率、外债、资本流动逆转、银行与企业资产负债表错配。
- 俄罗斯 1998：主权债务违约、油价冲击、卢布危机。
- 阿根廷 2001：货币局崩溃、债务违约、银行冻结存款。
- 美国 2008：房地产泡沫、影子银行、证券化杠杆、系统性金融危机。
- 欧债危机 2009 后：主权债务、银行资产负债表、财政紧缩与经济衰退循环。
- 津巴布韦、委内瑞拉、黎巴嫩、斯里兰卡：恶性通胀、银行体系崩溃、外汇危机与国家信用受损。

本框架的核心判断方法是：

```text
不要只看官方宏观表面数据；
不要只看网民情绪；
不要只看外资流向；
而是把官方数据、民间商业数据、外资数据、企业数据、债务数据、居民数据、就业数据、地方财政数据、产业链数据、贸易数据、社会压力数据和政策反应强度放在一起交叉验证。
```

## 1. 核心研究目标

本项目需要回答以下问题：

1. 中国当前是普通周期下行，还是资产负债表衰退？
2. 房地产风险是否已经从开发商端传导到居民、银行和地方财政？
3. 地方债与城投债风险是否正在形成“中国式欧债压力”？
4. 银行体系风险是显性可控，还是隐性坏账延后确认？
5. 居民是否正在进入防御性储蓄、降杠杆和消费降级？
6. 就业问题是局部行业调整，还是岗位质量系统性下降？
7. 民间投资和企业现金流是否支持长期增长？
8. 外资是否只是短期观望，还是正在重估中国资产和供应链地位？
9. 中国制造业增长是由真实竞争力驱动，还是部分行业靠产能过剩和价格战维持？
10. 中国是否存在货币危机、恶性通胀、外债危机风险？
11. 最可能的风险路径是急性金融危机，还是日本式慢性低增长？
12. 哪些指标可以作为未来 3–24 个月的预警信号？

## 2. 总体判断框架

中国当前最需要关注的不是单一 GDP 增速，而是以下链条是否形成闭环：

```text
房地产卖不掉
→ 居民资产缩水
→ 居民减少消费和贷款
→ 企业利润和现金流下降
→ 就业、工资和民间投资承压
→ 地方财政和土地收入下降
→ 城投债、地方平台和银行资产质量承压
→ 信贷扩张意愿下降
→ 经济进入低信心、低投资、低通胀、低回报循环
```

如果这条链条被多源数据持续验证，中国当前风险就不是普通周期波动，而是：

```text
房地产资产负债表衰退
+ 地方财政/城投债压力
+ 银行隐性坏账延后确认
+ 居民消费降级
+ 民间投资信心不足
+ 外资信心降级
+ 供应链再配置
```

## 3. 历史危机模板与中国当前相似度

| 历史模板 | 核心机制 | 中国相似点 | 中国不同点 | 初步风险判断 |
|---|---|---|---|---|
| 日本 1990s | 房地产泡沫破裂、资产负债表衰退、长期通缩 | 房地产下行、居民谨慎、企业投资弱、低通胀压力 | 中国制造业和出口仍强，政府控制金融系统能力更强 | 高度相似，最需要警惕 |
| 欧债危机 | 主权债务、银行、财政紧缩循环 | 地方债、城投、土地财政、地方银行纠缠 | 中国有本币主权和中央协调能力，不是欧元区约束 | 中高风险 |
| 美国 2008 | 房地产泡沫、银行坏账、金融杠杆 | 房地产和银行资产质量压力 | 中国没有同等规模的次贷证券化和自由挤兑机制 | 局部相似 |
| 亚洲金融危机 1997 | 外债、固定汇率、资本外逃、外储不足 | 存在资本流出和汇率预期压力 | 中国外储高、经常账户顺差、资本账户未完全开放 | 经典货币危机风险较低 |
| 阿根廷 2001 | 固定汇率、美元债、债务违约、银行冻结 | 局部存在居民信心和资本外流担忧 | 中国不是美元化经济，外储和贸易顺差更强 | 低到中 |
| 魏玛/津巴布韦/委内瑞拉 | 财政赤字货币化、货币信用崩溃、恶性通胀 | 财政压力存在 | 当前更像需求不足和低通胀，不是物价失控 | 恶性通胀风险低 |
| 黎巴嫩 2019 | 银行体系内爆、存款冻结、汇率崩溃 | 居民对银行和资产安全可能有担忧 | 中国银行体系国有色彩强、资本管制强 | 极端情景，不是基准 |

## 4. 十三道数据支撑体系

### 第一道：官方宏观数据

官方数据不能完全相信，但必须纳入，因为它代表政策口径、官方叙事和基础宏观结构。

| 观察方向 | 关键指标 | 用途 | 风险信号 |
|---|---|---|---|
| GDP | 实际 GDP、名义 GDP、行业增速 | 看总量增长和结构分化 | 名义增速明显低于实际增速，显示价格压力 |
| 消费 | 社零、服务消费、餐饮、汽车、家电 | 看居民需求 | 社零低迷，耐用品和大额消费弱 |
| 投资 | 固定资产投资、民间投资、制造业投资、基建投资 | 看企业和政府扩张意愿 | 民间投资长期弱于国企/基建 |
| 房地产 | 开发投资、销售面积、销售额、新开工、竣工、到位资金 | 看地产链条压力 | 销售、融资、新开工、投资同步下滑 |
| 价格 | CPI、核心 CPI、PPI | 看通胀或通缩压力 | CPI 弱、PPI 负、核心通胀低 |
| 财政 | 一般公共预算、政府性基金、土地出让收入 | 看地方财政 | 土地财政下降，支出刚性上升 |
| 金融 | M2、社融、居民贷款、企业贷款、票据融资 | 看信贷需求 | M2 高但贷款需求弱，居民中长期贷款弱 |
| 外部 | 贸易顺差、经常账户、外储、外债 | 看外汇和外部安全 | 经常账户顺差收窄，外储下降，短债压力上升 |
| 人口 | 出生率、老龄化、劳动年龄人口 | 看长期需求 | 人口减少、老龄化加速、房地产长期需求下降 |

使用原则：

```text
官方数据用于确定宏观边界，不能单独作为事实结论。
若官方数据和民间数据冲突，以交易数据、现金流数据、资本流动数据做第二判断。
```

### 第二道：民间、商业与舆情大数据

这一道用于发现官方数据掩盖或滞后的真实压力。需要注意：网民评论只能作为症状雷达，不能直接作为事实。真正有价值的是交易数据、平台数据、商业数据和舆情热词的交叉验证。

| 方向 | 数据来源 | 观察重点 |
|---|---|---|
| 房地产 | 贝壳、安居客、中指、克而瑞、诸葛找房、阿里资产、京东资产、法拍平台 | 二手房挂牌量、降价比例、成交周期、法拍数量、流拍率 |
| 就业 | BOSS、智联、猎聘、前程无忧、脉脉、领英、地方人才网 | 岗位数量、薪资中位数、应届生岗位、裁员帖、外包化 |
| 消费 | 美团、大众点评、抖音团购、淘宝、京东、拼多多、Kantar、Bain、Worldpanel | 客单价、折扣强度、低价替代、品类降级 |
| 商业地产 | 高德、百度地图、企查查、天眼查、点评门店、商场招商 | 商铺转让、空置率、品牌撤店、租金下调 |
| 出行旅游 | 百度迁徙、高德拥堵、地铁客流、航班管家、携程、同程、酒店平台 | 人流、旅游客单价、酒店入住率、低价游比例 |
| 工业物流 | 满帮、G7、货拉拉、港口 AIS、集装箱指数、快递指数 | 货运量、运价、空驶率、港口吞吐、出口链景气 |
| 金融压力 | 城投债利差、信用债违约、票据利率、银行股、AMC 资产处置 | 地方债压力、融资困难、小银行风险 |
| 居民压力 | 法拍、提前还贷、断供帖、债务协商、消费贷逾期、黄金购买 | 居民现金流、资产缩水、避险倾向 |

重点舆情平台：

| 平台 | 适合观察内容 |
|---|---|
| 小红书 | 消费降级、房贷、失业、母婴、教育、旅游、城市生活成本 |
| 抖音/快手 | 下沉市场、实体店、工厂、工地、外卖骑手、货车司机 |
| B站 | 年轻人就业、考公考研、行业衰退、消费心理 |
| 知乎 | 中产、房产、职业、移民、投资、宏观讨论 |
| 微博 | 突发事件、裁员、楼盘维权、金融风险 |
| 脉脉 | 互联网裁员、薪资、职场变化 |
| 雪球/东方财富 | 股民预期、地产股、银行股、城投债风险 |
| 贴吧/本地论坛 | 三四线房价、工资、工厂、商铺、地方财政感受 |

关键词体系：

```text
房地产：房子卖不掉、降价卖房、断供、法拍房、二手房挂牌、没人接盘、烂尾楼、保交楼、提前还贷、负资产、房贷压力、租金下降、商铺卖不掉

就业：裁员、降薪、优化、毕业即失业、35岁危机、找不到工作、外包、灵活就业、跑外卖、开网约车、考公、考编、考研二战、失业金、社保断缴

消费：消费降级、不敢花钱、只买刚需、拼多多、平替、折扣店、临期食品、二手闲置、不买房不买车、存钱、金价、穷游、9.9团购

地方财政：公务员降薪、教师降薪、医院欠薪、城投、工程款拖欠、地方债、绩效停发、奖金取消、平台公司、政府项目停工

企业经营：关店、转让、倒闭、招商困难、租金下降、订单减少、工厂放假、没有加班、回款难、账期拉长、降本增效
```

使用原则：

```text
单条帖子不作为证据；
单个平台热度不作为结论；
只有同一主题在多个平台、多个城市、多个行业同步升温，并能被商业数据验证，才进入报告结论。
```

### 第三道：外资、资本流动与跨国公司数据

外资是高度敏感的数据，但也不是完全没有偏差。香港、开曼、新加坡通道资金，内资绕道回流，利润再投资，跨国公司内部贷款，汇率估值和一次性并购都会干扰数据。因此必须多口径交叉验证。

| 指标组 | 看什么 | 解释 |
|---|---|---|
| 直接投资 FDI | 外资新设、增资、利润再投资、股权投资、外商实际使用金额 | 长期资本是否还愿意进入中国 |
| 国际收支口径 FDI | SAFE BOP 里的 direct investment liabilities/assets | 更接近跨境资金净流动 |
| 证券投资 FPI | 外资买卖 A 股、港股中国资产、人民币债券 | 最敏感，跑得最快 |
| 外国企业再投资 | 是否把中国利润留在中国继续扩张 | 比口头表态更真实 |
| 外企调查 | AmCham、欧盟商会、JETRO、德国商会、英国商会 | 企业主观信心和未来投资计划 |
| 跨国公司财报 | Apple、Tesla、VW、BMW、LVMH、Nike、Starbucks 等 | 中国收入、利润、资本开支和门店计划 |
| 供应链迁移 | China+1、越南、印度、墨西哥、东盟新增投资 | 新增产能是否转移 |
| 中国对外投资 | 中国企业对外 FDI、海外并购、海外建厂 | 中国资本是否也在全球配置 |

风险评级：

| 指标 | 正常 | 警戒 | 危险 |
|---|---|---|---|
| FDI 流入 | 小幅波动 | 连续下降 | 连续两位数下降且大项目减少 |
| BOP 直接投资净额 | 正 | 接近零 | 持续为负 |
| 证券投资 | 有进有出 | 外资低配 | 股票债券同时净流出 |
| 外企调查 | 谨慎但扩张 | 维持存量 | 明确转移新增投资 |
| 外资债券持仓 | 稳定 | 缓慢下降 | 快速下降 |
| 跨国公司 capex | 持平 | 不再扩张 | 关闭、出售、迁移 |
| 供应链 | China+1 | 新增产能分散 | 明确去中国化 |
| 利润再投资 | 留在中国 | 部分汇出 | 大量汇出、不再增资 |

阶段判断：

```text
外资没有全面离开中国，不等于外资信心没有下降；
外企仍在中国赚钱，不等于愿意继续新增重仓；
更关键的是：存量是否保留，增量是否转移，利润是否再投资。
```

### 第四道：企业利润与现金流数据

企业有没有问题，最终会反映在利润率、现金流、应收账款、库存、裁员和资本开支上。

| 观察方向 | 数据来源 | 看什么 |
|---|---|---|
| 工业企业利润 | 国家统计局、行业协会 | 利润率、亏损面、库存 |
| 上市公司财报 | A 股、港股、美股中概股 | 收入、毛利率、净利率、现金流 |
| 民企财报 | 新三板、债券发行人、企业年报 | 是否降本、裁员、压缩投资 |
| 应收账款 | 上市公司财报、建筑/制造企业公告 | 回款是否变慢 |
| 经营现金流 | 财报 | 是否账面盈利但没有现金 |
| 企业注销/吊销 | 企查查、天眼查 | 小微企业退出潮 |
| 破产重整 | 全国企业破产重整案件信息网 | 哪些行业集中破产 |

重点判断：

```text
收入增长但利润下降 = 价格战或成本压力；
利润为正但经营现金流恶化 = 回款问题；
应收账款增长快于收入 = 赊销和账期压力；
库存增长快于销售 = 需求不足或产能过剩；
资本开支下降 = 企业对未来不乐观。
```

### 第五道：价格体系与通缩压力数据

官方 CPI/PPI 只是表面，真实价格链条更重要。

| 观察方向 | 数据来源 | 风险信号 |
|---|---|---|
| PPI | 工业品出厂价格 | 企业端价格压力 |
| 核心 CPI | 食品、服务、租金、耐用品 | 居民消费强弱 |
| 房租 | 贝壳、自如、链家、58 | 年轻人收入和城市需求 |
| 工业品价格 | 钢铁、水泥、玻璃、化工、锂电材料 | 基建、地产、制造景气 |
| 农产品价格 | 猪肉、蔬菜、水果、粮食 | 居民生活成本和农民收入 |
| 汽车价格 | 终端折扣、库存、二手车残值 | 价格战和产能过剩 |
| 电商价格 | 拼多多、淘宝、京东、1688 | 低价内卷程度 |

需要区分：

```text
通胀：居民抢货、价格普涨、工资跟涨；
通缩：需求不足、企业降价、工资不涨、居民延迟消费；
结构性价格战：某些行业产能过剩，价格下降但总量还在增长；
输入型成本压力：上游涨价但下游无法涨价，企业利润被挤压。
```

### 第六道：债务与信用压力数据

金融风险往往比 GDP 更早反映在债务利差、违约、展期、票据利率、银行净息差和不良迁徙上。

| 观察方向 | 数据来源 | 风险信号 |
|---|---|---|
| 社融结构 | 央行 | 信贷是否靠政府和票据支撑 |
| 居民贷款 | 央行、银行财报 | 居民是否停止加杠杆 |
| 企业债利差 | Wind、债券市场 | 民企、城投、地产债压力 |
| 违约债券 | 债券公告、财新、DM | 哪些主体违约 |
| 非标展期 | 信托、城投、地方新闻 | 隐性债务压力 |
| 票据利率 | 票据市场 | 企业真实融资需求 |
| 银行净息差 | 银行财报 | 银行利润承压程度 |
| 不良贷款迁徙率 | 银行财报 | 坏账是否正在形成 |
| 拨备覆盖率 | 银行财报 | 银行有没有足够缓冲 |

核心问题：

```text
债务能不能滚得动？
利息能不能付？
银行愿不愿意继续贷？
坏账是被真实确认，还是被展期、置换、AMC、隐性担保延后？
```

### 第七道：居民资产负债表数据

中国当前最关键的问题之一，是居民部门是否进入资产负债表修复。

| 观察方向 | 数据来源 | 风险信号 |
|---|---|---|
| 居民存款 | 央行 | 高储蓄、不消费 |
| 居民贷款 | 央行 | 不买房、不消费贷 |
| 房贷提前还款 | 银行、社媒、论坛 | 居民主动降杠杆 |
| 消费贷逾期 | 银行、黑猫投诉、裁判文书 | 现金流恶化 |
| 信用卡逾期 | 银行财报、投诉平台 | 中低收入压力 |
| 法拍房 | 阿里资产、京东资产 | 断供和抵押物处置 |
| 二手平台交易 | 闲鱼、转转 | 卖闲置、消费降级 |
| 黄金购买 | 世界黄金协会、金店销售 | 避险储蓄上升 |

判断规则：

```text
居民存款上升 + 贷款下降 + 提前还贷 + 消费贷逾期 + 二手交易上升 + 黄金购买上升
= 居民进入防御状态。
```

### 第八道：就业质量数据

官方失业率不足以判断真实就业压力。需要看岗位数量、薪资质量、正式岗位比例和灵活就业饱和度。

| 观察方向 | 数据来源 | 看什么 |
|---|---|---|
| 招聘岗位数 | BOSS、智联、猎聘、前程无忧 | 岗位是否减少 |
| 薪资中位数 | 招聘平台 | 是否降薪 |
| 应届生岗位 | 校招平台、牛客、应届生求职网 | 青年就业 |
| 灵活就业收入 | 外卖、网约车、货拉拉司机论坛 | 兜底岗位是否饱和 |
| 社保缴纳人数 | 地方人社、企业年报 | 正式就业质量 |
| 公务员/事业单位降薪 | 地方论坛、小红书、微博 | 地方财政传导 |
| 裁员舆情 | 脉脉、知乎、小红书 | 行业压力 |
| 加班强度 | 工厂、互联网、服务业评论 | 订单和用工需求 |

重点不是“有没有工作”，而是：

```text
工资有没有下降？
岗位有没有减少？
正式岗位有没有被临时工、外包、灵活就业替代？
灵活就业是否越来越卷？
应届生是否被迫考公、考研、延迟就业？
```

### 第九道：地方财政真实压力数据

中国很多风险藏在地方政府、地方平台和土地财政里。

| 观察方向 | 数据来源 | 风险信号 |
|---|---|---|
| 土地出让收入 | 财政部、地方财政报告 | 土地财政塌陷 |
| 政府性基金收入 | 财政报告 | 地方可支配财力下降 |
| 地方专项债项目收益 | 债券说明书、审计报告 | 项目现金流不足 |
| 城投债利差 | 债券市场 | 地方信用分化 |
| 城投非标展期 | 信托、地方平台、债市信息 | 隐债压力 |
| 公务员/教师/医生降薪 | 本地论坛、小红书 | 财政压力向工资传导 |
| 工程款拖欠 | 建筑企业、裁判文书、抖音 | 地方政府现金流紧张 |
| 政府采购减少 | 政采网 | 公共部门收缩 |

高风险组合：

```text
土地收入下降
+ 城投利差上升
+ 工程款拖欠
+ 公职人员降薪
+ 政府采购收缩
= 地方财政压力真实化。
```

### 第十道：产业链与产能过剩数据

中国制造业强，但需要判断这种强是来自真实竞争力，还是产能过剩和价格战。

| 行业 | 重点指标 |
|---|---|
| 新能源车 | 库存、终端折扣、单车毛利、出口依赖 |
| 锂电池 | 产能利用率、碳酸锂价格、企业亏损面 |
| 光伏 | 组件价格、产能利用率、企业现金流 |
| 钢铁 | 粗钢产量、钢价、钢厂利润 |
| 水泥 | 出货率、价格、库存 |
| 化工 | 价差、开工率、库存 |
| 家电 | 出口、内销、价格战 |
| 半导体 | 国产替代进度、库存周期、设备订单 |

判断规则：

```text
产量增长 + 价格下降 + 利润下降 + 库存上升 = 产能过剩；
出口增长 + 出口价格下降 + 企业利润下降 = 靠降价抢市场；
技术升级 + 毛利率稳定 + 全球份额提升 = 真实竞争力增强。
```

### 第十一道：国际贸易真实竞争力数据

外贸不能只看出口金额，因为出口金额可能靠降价撑住。

| 观察方向 | 数据来源 | 风险信号 |
|---|---|---|
| 出口数量 vs 出口金额 | 海关、UN Comtrade | 数量涨、金额不涨，说明降价 |
| 出口价格指数 | 海关、WTO、行业数据 | 出口价格下行 |
| 主要市场份额 | 美国、欧盟、东盟海关 | 是否被越南、印度、墨西哥替代 |
| 贸易摩擦 | 反倾销、关税、制裁 | 外需不确定性 |
| 集装箱运价 | SCFI、Freightos | 外贸景气 |
| 港口吞吐 | 港口数据、AIS | 实际物流 |
| 义乌指数 | 小商品出口 | 中小外贸景气 |
| 跨境电商数据 | 亚马逊、Temu、Shein、阿里国际站 | 低价竞争和利润压力 |

核心问题：

```text
出口增长是技术、品牌、效率带来的高质量增长，
还是低价、补贴、产能过剩和利润压缩带来的数量增长？
```

### 第十二道：社会压力与公共服务数据

社会压力数据不能情绪化使用，但可以作为经济压力外溢的后验信号。

| 观察方向 | 数据来源 | 风险信号 |
|---|---|---|
| 欠薪投诉 | 人社、黑猫、微博、抖音 | 企业现金流恶化 |
| 劳动仲裁 | 裁判文书、地方仲裁数据 | 劳资纠纷增加 |
| 破产案件 | 法院、破产重整平台 | 企业出清加速 |
| 民间借贷纠纷 | 裁判文书 | 居民/小微资金紧张 |
| 婚育数据 | 民政、医院、社媒 | 长期信心下降 |
| 医保/社保缴费压力 | 地方政策、社媒 | 公共财政压力 |
| 教培/留学/移民咨询 | 小红书、知乎、机构数据 | 中产信心变化 |

使用原则：

```text
社会压力数据不能直接证明宏观危机；
但如果与就业、消费、居民负债、地方财政同时恶化，可以作为风险外溢证据。
```

### 第十三道：政策反应强度

政策本身会泄露压力。政府未必承认所有问题，但政策力度、频率和方向可以反映真实压力。

| 政策动作 | 可能说明 |
|---|---|
| 连续降准降息 | 信用需求弱，经济需要托底 |
| 存量房贷利率下调 | 居民房贷压力和消费压力大 |
| 地方债置换 | 地方债压力已经显性化 |
| 特别国债 | 常规财政空间不足 |
| 保交楼专项资金 | 房地产交付风险高 |
| 放松限购限贷 | 楼市需求不足 |
| 鼓励生育补贴 | 人口压力成为政策重点 |
| 消费券和以旧换新 | 居民消费不足 |
| 鼓励民企 | 民间投资信心不足 |
| 加强资本流动管理 | 汇率和资本外流压力 |

判断原则：

```text
政策不是数据，但政策强度是信号。
如果救房地产、救地方债、救消费、救就业、救民企、稳外资政策密集出台，说明压力已经足够高。
```

## 5. 核心风险链条

### 5.1 房地产链条

```text
房价预期下降
→ 二手房挂牌增加
→ 成交周期拉长
→ 法拍和流拍增加
→ 开发商回款下降
→ 新开工下降
→ 土地出让下降
→ 地方财政承压
→ 城投和地方银行压力上升
```

重点监测：

- 商品房销售面积和销售额。
- 二手房挂牌量。
- 成交周期。
- 降价房源比例。
- 法拍房数量和流拍率。
- 银行处置抵押房产。
- 居民中长期贷款。
- 土地出让收入。

### 5.2 居民资产负债表链条

```text
房产财富缩水
→ 居民减少大额消费
→ 提前还贷或不贷款
→ 存款增加
→ 消费降级
→ 企业收入和利润承压
→ 就业和工资预期下降
→ 居民继续防御
```

重点监测：

- 居民存款。
- 居民短期贷款和中长期贷款。
- 房贷提前还款。
- 消费贷和信用卡逾期。
- 社零结构。
- 快消均价。
- 二手交易和黄金购买。

### 5.3 地方财政与城投链条

```text
土地收入下降
→ 政府性基金收入下降
→ 地方项目资金不足
→ 工程款拖欠
→ 城投债务滚续压力上升
→ 地方平台展期或置换
→ 地方银行资产质量承压
→ 公共服务和财政支出收缩
```

重点监测：

- 土地出让收入。
- 地方政府性基金收入。
- 城投债利差。
- 非标展期。
- 工程款拖欠。
- 政府采购减少。
- 公务员、教师、医生降薪舆情。

### 5.4 银行风险链条

```text
地产和城投资产质量下降
→ 银行不良贷款压力上升
→ 展期、置换、AMC 处置延后确认
→ 银行净息差下降
→ 拨备消耗
→ 小银行资本压力上升
→ 信贷投放更加谨慎
```

重点监测：

- 银行净息差。
- 不良率和关注类贷款。
- 不良迁徙率。
- 拨备覆盖率。
- 地产贷款占比。
- 城投和地方平台敞口。
- 小银行资本充足率。
- 银行处置抵押物折价情况。

### 5.5 外资和资本信心链条

```text
国内需求走弱
+ 地缘政治风险上升
+ 政策不确定性
→ 外企新增投资谨慎
→ 利润再投资下降
→ 证券投资流出
→ 供应链新增产能转向海外
→ 中国从全球增量中心变成存量市场
```

重点监测：

- BOP 直接投资负债。
- 实际使用外资。
- 外资证券投资。
- 外资持有人民币债券。
- 外企商会调查。
- 跨国公司中国区收入和 capex。
- 越南、印度、墨西哥、东盟新增 FDI。
- 中国企业对外投资。

### 5.6 产能过剩和价格战链条

```text
内需不足
→ 企业依赖出口和降价
→ 价格战加剧
→ 毛利率下降
→ 现金流恶化
→ 行业整合或倒闭
→ 就业和投资继续承压
```

重点监测：

- 新能源车价格和库存。
- 光伏组件价格。
- 锂电池产能利用率。
- 钢铁、水泥、化工价格。
- 出口数量与出口金额背离。
- 企业毛利率和现金流。

## 6. 三角验证规则

任何重要结论必须满足至少三类证据。

### 示例：判断“消费降级加重”

不能只靠小红书帖子，需要至少满足以下三项：

1. 社交平台中“消费降级、平替、存钱、不敢花钱”等讨论升温。
2. 商业数据中低价渠道、折扣店、O2O、团购增长。
3. 快消数据中销量增长但销售额不涨，均价下降。
4. 美团/抖音团购中低价套餐占比上升。
5. 高端消费、奢侈品、餐饮、商场客流承压。

### 示例：判断“房地产流动性枯竭”

需要同时观察：

1. 二手房挂牌量增加。
2. 成交周期拉长。
3. 成交价低于挂牌价且多轮降价。
4. 法拍房增加。
5. 法拍流拍率上升。
6. 银行处置抵押房产增多且折价仍难卖。
7. 居民中长期贷款弱。

### 示例：判断“地方财政真实承压”

需要同时观察：

1. 土地出让收入下降。
2. 政府性基金收入下降。
3. 城投债利差扩大。
4. 工程款拖欠增加。
5. 政府采购收缩。
6. 公职人员奖金、绩效、津贴下降舆情增多。
7. 地方平台债务置换、展期增多。

## 8. 未来情景推演

### 情景 A：基准情景——慢修复

特征：

- GDP 仍维持正增长。
- 制造业、出口、高科技产业继续托底。
- 房地产继续出清。
- 地方债通过置换、展期、低息化慢慢处理。
- 居民消费恢复缓慢。
- 通胀低位，企业利润承压。
- 金融风险不集中爆发，但长期压低增长效率。

对应历史模板：

```text
日本式资产负债表衰退 + 中国式政策托底
```

### 情景 B：下行情景——房地产和地方债共振

触发条件：

- 房地产销售继续两位数下滑。
- 房价预期继续恶化。
- 开发商资金链进一步收缩。
- 土地收入继续下降。
- LGFV 债务展期压力传导到地方银行。
- 出口遇到外部冲击。
- 居民消费和民间投资继续弱。

可能结果：

```text
低通胀、低信心、低投资、银行利润下降、地方财政紧缩。
```

### 情景 C：上行情景——房地产止跌与资产负债表修复

触发条件：

- 保交楼和存量房去化真正见效。
- 房地产销售跌幅明显收窄。
- 地方债/LGFV 形成透明、可信的重组框架。
- 银行资产质量更透明。
- 政策从投资托底转向居民收入、社保、消费和民企信心。
- 外需没有大幅恶化。

可能结果：

```text
中国避免日本式长期停滞，逐步进入低房地产依赖、高端制造、服务消费的新平衡。
```

## 11. 初步总判断

当前中国最需要防范的不是典型外债危机、恶性通胀或短期货币崩盘，而是：

```text
日本式资产负债表衰退
+ 中国式地方债和城投风险
+ 房地产长期出清
+ 银行隐性坏账延后确认
+ 居民消费降级
+ 民间投资信心不足
+ 外资增量转移
+ 制造业价格战和产能过剩
```

短期看，中国仍有外汇储备、经常账户顺差、政策控制力、国有银行体系、资本管制和制造业能力作为缓冲。

长期看，如果房地产、居民资产负债表、地方财政、银行资产质量、民间投资、就业质量和外资信心同时恶化，中国可能进入长期低信心、低通胀、低投资、低回报的慢性循环。

真正需要持续验证的核心问题是：

```text
中国是在完成一次痛苦但可控的结构转型，
还是正在进入长期资产负债表衰退？
```

本框架后续所有调研都应围绕这个问题展开。
