# 08｜企业、店铺与工程项目活跃度监测框架

> 本文档补充实体经济活跃度监测：新开公司、注销/吊销公司、店铺新开/闭店、同址换店率、工程项目发标量、项目资金来源。它们能回答“经济好的时候是什么样，现在是什么样”。

---

## 1. 为什么必须加入

官方 GDP、社零、投资数据可能滞后或被总量掩盖。企业、店铺、工程项目是实体经济的微观行为数据。

它们直接反映：

1. 创业和经营信心。
2. 小微企业生存状态。
3. 线下消费和商圈活力。
4. 投资项目真实启动情况。
5. 地方财政和城投托底强度。
6. 民间资本是否愿意进入新项目。

---

## 2. 经济好时 vs 经济弱时

| 指标 | 经济好时 | 经济弱时 |
|---|---|---|
| 新设企业 | 上升，行业分布扩散 | 下降，集中在低成本或政策补贴行业 |
| 注销/吊销企业 | 稳定或低位 | 上升，尤其小微和服务业 |
| 净新增企业 | 持续为正 | 收窄甚至转负 |
| 经营异常 | 低位 | 上升，说明现金流、地址、年报、合规压力增大 |
| 破产重整 | 低位 | 上升，行业集中度增强 |
| 新开店 | 商圈扩张，品类丰富 | 新开减少，低价业态增加 |
| 闭店 | 低位 | 上升，餐饮、零售、教培、服务业先反映 |
| 同址换店率 | 低，店铺稳定经营 | 高，说明生意不稳定、租金承压、试错频繁 |
| 商铺转让 | 少 | 多，且转让周期拉长 |
| 工程发标量 | 稳定增长，民间项目多 | 下降或依赖政府/国资托底 |
| 项目资金来源 | 民资、市场化资金活跃 | 财政、专项债、城投、国资占比上升 |
| 流标/废标 | 低 | 上升，说明资金、预算、施工利润或回款风险高 |
| 工程款拖欠 | 低 | 上升，向建筑企业现金流传导 |

## 3. 新增数据源

### 3.1 企业新设/注销/吊销/破产

数据源 ID：

```text
COMPANY_REGISTRATION_EXIT
```

观察指标：

- 新设企业数量。
- 注销企业数量。
- 吊销企业数量。
- 经营异常企业数量。
- 破产重整案件数量。
- 净新增企业。
- 注销率。
- 行业、城市、注册资本、企业年龄分布。

候选来源：

- 国家企业信用信息公示系统。
- 国家市场监督管理总局。
- 地方市场监管局。
- 全国企业破产重整案件信息网。
- 企查查、天眼查、爱企查等企业信息服务。

### 3.2 店铺新开/闭店/同址换店率

数据源 ID：

```text
STORE_ADDRESS_TURNOVER
```

观察指标：

- 新开店数量。
- 闭店数量。
- 净新增店铺。
- 同址换店率。
- 空置率。
- 商铺转让率。
- 商铺租金。
- 评论新增量、团购活跃度。

候选来源：

- 高德地图、百度地图、腾讯地图 POI。
- 大众点评、美团、抖音本地生活。
- 商铺租赁平台。
- 企查查、天眼查、国家企业信用信息公示系统。
- 商场招商和品牌撤店信息。

### 3.3 工程招投标与资金来源

数据源 ID：

```text
PROJECT_TENDER_ACTIVITY
```

观察指标：

- 发标项目数量。
- 预算金额。
- 中标金额。
- 流标/废标率。
- 项目延期率。
- 项目取消率。
- 资金来源结构。
- 国资/财政/城投/专项债/民间资金占比。

候选来源：

- 中国政府采购网。
- 中国招标投标公共服务平台。
- 全国公共资源交易平台。
- 各省市公共资源交易中心。
- 地方发改委、财政、住建、交通、水利公告。
- 上市建筑企业订单公告。

## 4. 新增模块

### 4.1 企业与店铺新陈代谢

模块 ID：

```text
entity_activity
```

核心判断：

```text
新设减少 + 注销吊销上升 + 经营异常上升 + 闭店上升 + 同址换店率上升
= 小微经营压力和线下消费压力上升。
```

### 4.2 工程项目与投资活跃度

模块 ID：

```text
project_activity
```

核心判断：

```text
发标量下降 + 民资项目占比下降 + 国资/财政/城投项目占比上升 + 流标废标上升 + 工程款拖欠上升
= 投资活跃度下降且政府托底占比提高。
```

## 5. 历史对比方法

每个城市、行业、资金类型都需要建立历史基准：

```text
当前值
vs 上月
vs 去年同期
vs 近 3 年均值
vs 近 5 年均值
vs 疫情前均值
vs 房地产高峰期均值
```

重点不是单点数据，而是趋势：

```text
企业净新增是否持续下降？
注销率是否超过历史正常波动？
同址换店率是否持续上升？
工程发标是否从民资转向国资托底？
流标废标和工程款拖欠是否同步上升？
```
