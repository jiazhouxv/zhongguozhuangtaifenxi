# 10｜V0.1.5 移动端、圆环、时间节点与数据源抽屉修复说明

## 1. 本轮修复目标

根据页面截图和使用反馈，V0.1.5 修复以下问题：

1. 顶部总风险圆环在窄屏下变形。
2. 左右场景切换按钮在窄屏下消失。
3. 触摸屏无法通过手指左右滑动切换场景。
4. 中国当前状态缺少类似日本 1990 的时间节点。
5. 高风险模块缺少醒目的红色预警符号。
6. 数据源控制台不应占用主页面，应改成侧边可折叠抽屉。

---

## 2. 圆环变形修复

修复方式：

```css
.ring {
  width: 168px;
  aspect-ratio: 1 / 1;
  height: auto;
  min-width: 168px;
  flex: 0 0 auto;
}

.miniRing {
  width: 78px;
  aspect-ratio: 1 / 1;
  height: auto;
  min-width: 78px;
  flex: 0 0 auto;
}
```

这样在窄屏、flex 布局和响应式压缩时，圆环不会被拉成长椭圆。

---

## 3. 移动端左右切换修复

之前 V0.1.4 在窄屏下隐藏了左右按钮。V0.1.5 改为：

```css
.navbtn {
  display: inline-block;
}
```

移动端依然保留左右按钮。

---

## 4. 触摸滑动切换

V0.1.5 在顶部场景卡片上增加触摸事件：

```text
手指左滑 → 下一个场景
手指右滑 → 上一个场景
```

只在横向位移明显大于纵向位移时触发，避免普通上下滚动误触。

---

## 5. 中国时间节点

新增：

```text
public/data/china_timeline.json
```

当前包含样例节点：

```text
2021 年底
2022 年底
2023 年底
2024 年底
2025 年底
2026-06-30 当前
```

中国当前场景现在也支持顶部节点下拉框。当前节点 `2026-06-30` 使用 dashboard 实时样例分数；历史节点为样例框架，后续接入真实历史数据后替换。

---

## 6. 红色预警感叹号

风险分数 >= 75 时显示红色感叹号：

```text
!
```

用于：

- 总风险等级。
- Top 6 风险驱动。
- 风险模块标题。

---

## 7. 数据源控制台改为左侧抽屉

V0.1.5 将数据源控制台从主页面移出，改为左侧固定按钮：

```text
数据源
```

点击后打开左侧抽屉，再点关闭或点击遮罩隐藏。

保留功能：

- 启用 / 禁用。
- 显示 / 隐藏。
- 是否纳入评分。
- 权重滑块。
- 数据截至时间。
- 来源说明。
