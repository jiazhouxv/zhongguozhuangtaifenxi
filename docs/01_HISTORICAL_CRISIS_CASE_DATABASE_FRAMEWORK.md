# 01｜历史国家级危机案例数据库框架

> 本文档用于建立“历史危机案例库”，作为中国状态分析的长期比较基准。当前版本先搭建框架，后续逐步扩充每个国家/地区的结构化案例、关键指标、触发机制、政策应对、结果和与中国当前状态的可比性。

---

## 1. 建库目的

中国当前状态不能只靠中国自身数据判断。必须把历史上各国经历过的经济大萧条、金融危机、银行危机、货币大贬值、恶性通胀、主权债务危机、房地产泡沫破裂和资产负债表衰退作为参照系。

历史案例库用于回答：

1. 中国当前更像哪类历史危机？
2. 哪些风险链条已经出现，哪些还没有出现？
3. 哪些指标是危机前的领先信号？
4. 哪些政策能止血，哪些政策会延缓出清？
5. 当前风险更可能是急性崩盘，还是慢性资产负债表衰退？
6. 中国与历史案例最大的不同点是什么？

## 2. 危机类型分类

### 2.1 大萧条 / 深度衰退

定义：

```text
GDP 多年大幅下滑，失业率显著上升，需求、投资、信贷和社会信心同步恶化。
```

典型案例：

- 美国 1929–1933 大萧条。
- 德国 1929–1933 大萧条。
- 英国、加拿大、澳大利亚 1930s 大萧条。
- 希腊 2009–2016 欧债危机期间深度衰退。
- 俄罗斯 1990s 转型萧条。
- 委内瑞拉 2013 后经济崩溃。

### 2.2 银行危机 / 金融危机

定义：

```text
银行体系出现严重挤兑、坏账、资本不足、流动性枯竭或大规模政府救助。
```

典型案例：

- 美国 1907 金融恐慌。
- 美国 1929–1933 银行危机。
- 日本 1990s 银行坏账危机。
- 北欧 1990s 银行危机：瑞典、芬兰、挪威。
- 亚洲金融危机 1997：泰国、印尼、韩国、马来西亚。
- 美国 2007–2009 次贷危机。
- 冰岛 2008 银行业崩溃。
- 爱尔兰 2008 后银行危机。
- 黎巴嫩 2019 后银行体系崩溃。

### 2.3 货币危机 / 汇率危机

定义：

```text
本币发生大幅贬值，固定汇率或准固定汇率失败，外储被快速消耗，资本外逃加剧。
```

典型案例：

- 墨西哥 1994 比索危机。
- 泰国 1997 泰铢危机。
- 印尼 1997–1998 卢比危机。
- 韩国 1997 韩元危机。
- 俄罗斯 1998 卢布危机。
- 土耳其 2001、2018 后里拉危机。
- 阿根廷 2001–2002 比索脱钩美元。
- 斯里兰卡 2022 外汇危机。

### 2.4 主权债务危机

定义：

```text
中央政府或准主权主体无法按原合同偿还债务，发生违约、重组、展期或大规模救助。
```

典型案例：

- 拉美债务危机 1982 后：墨西哥、阿根廷、巴西等。
- 俄罗斯 1998 国债违约。
- 阿根廷 2001 主权违约。
- 希腊 2009 后主权债务危机。
- 葡萄牙、爱尔兰、西班牙、塞浦路斯欧债危机。
- 斯里兰卡 2022 主权违约。

### 2.5 恶性通胀 / 货币信用崩溃

定义：

```text
货币购买力快速崩溃，经典门槛通常为月通胀率超过 50%。
```

典型案例：

- 德国魏玛 1921–1923。
- 匈牙利 1945–1946。
- 希腊 1943–1944。
- 中国 1940s 法币/金圆券恶性通胀。
- 玻利维亚 1984–1985。
- 秘鲁 1988–1990。
- 尼加拉瓜 1980s。
- 津巴布韦 2007–2009。
- 委内瑞拉 2016 后。

### 2.6 房地产泡沫破裂 / 资产负债表衰退

定义：

```text
房地产和资产价格下跌导致居民、企业、银行资产负债表受损，经济主体转向降杠杆和防御性储蓄。
```

典型案例：

- 日本 1990s。
- 瑞典、芬兰、挪威 1990s 房地产和银行危机。
- 美国 2007–2009 次贷危机。
- 爱尔兰 2008 后房地产泡沫破裂。
- 西班牙 2008 后房地产与银行危机。
- 中国当前需要重点对照。

## 5. 每个案例需要收集的数据

### 5.1 宏观数据

- 实际 GDP。
- 名义 GDP。
- 人均 GDP。
- 工业产出。
- 固定资产投资。
- 消费。
- 失业率。
- 工资。
- 通胀。
- PPI 或批发物价。

### 5.2 金融数据

- 银行数量和倒闭数量。
- 银行不良资产。
- 信贷增速。
- 利率。
- 银行存款。
- 股市指数。
- 房价。
- 债券收益率。
- 主权利差。

### 5.3 外部数据

- 经常账户。
- 外汇储备。
- 外债。
- 短期外债。
- 汇率。
- 资本流入/流出。
- 出口和进口。
- 商品价格。

### 5.4 财政与债务

- 政府收入。
- 政府支出。
- 财政赤字。
- 公共债务/GDP。
- 国内债务。
- 外债。
- 债务违约或重组。

### 5.5 社会与政治

- 失业。
- 贫困。
- 社会动荡。
- 政府更替。
- 政策制度变化。
- 存款冻结、资本管制、货币改革。

## 6. 与中国当前对照的核心问题

每个案例最终都要回答以下问题：

1. 该案例最核心的危机机制是什么？
2. 危机爆发前 1–3 年有哪些领先指标？
3. 该案例与中国当前哪个风险板块相似？
4. 中国是否已经出现相同指标？
5. 中国有哪些缓冲垫？
6. 中国有哪些特殊风险？
7. 如果按该案例推演，中国最坏路径是什么？
8. 哪些政策能避免重演？

## 9. 历史案例库与最终 HTML 可视化的关系

历史案例库不是静态背景材料，而是未来可视化页面中的“比较基准层”。

未来 HTML 页面可支持：

- 选择中国当前指标。
- 选择历史案例模板。
- 显示中国当前指标与历史案例危机前指标的相似度。
- 隐藏或显示某个案例。
- 调整案例权重。
- 显示“日本式风险”“欧债式风险”“亚洲金融危机式风险”“恶性通胀式风险”的相似度分数。

示例：

```text
中国当前房地产流动性指标
+ 居民贷款指标
+ 银行净息差指标
+ 通胀指标
→ 与日本 1990s 相似度上升

中国当前外储、外债、汇率、资本流动指标
→ 与亚洲金融危机 1997 相似度较低
```
