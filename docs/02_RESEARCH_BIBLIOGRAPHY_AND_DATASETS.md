# 02｜推荐学习书目、论文、研究报告与数据集

> 本文档整理国家级经济危机、金融危机、银行危机、货币危机、主权债务危机、恶性通胀、房地产泡沫和资产负债表衰退研究资料。后续每份资料都可以进入项目的 `sources` 数据库，用于支撑历史案例库和中国状态分析。

---

## 1. 总体学习路线

建议按照以下顺序学习：

```text
第一层：金融危机总框架
第二层：银行危机与信贷周期
第三层：大萧条与债务通缩
第四层：货币危机与资本流动
第五层：主权债务危机
第六层：恶性通胀
第七层：房地产泡沫与资产负债表衰退
第八层：亚洲金融危机、2008 全球金融危机、欧债危机等案例
第九层：替代数据、实时监测和经济 nowcasting
```

## 10. 替代数据、nowcasting 和实时监测方法

| 方向 | 可借鉴内容 |
|---|---|
| 移动定位数据 | 城市活动、人流、旅游、商圈热度、通勤恢复 |
| 企业注册/注销数据 | 小微企业活跃度、产业地理变化、城市经济活力 |
| 招聘平台数据 | 岗位数量、薪资中位数、行业用工需求 |
| 房产挂牌和成交数据 | 房地产真实流动性、成交周期、降价比例 |
| 法拍数据 | 居民和企业债务压力、银行抵押物处置 |
| 电商和团购数据 | 消费降级、低价渠道、客单价变化 |
| 货运与港口数据 | 工业和出口真实活动 |
| 新闻与舆情文本 | 风险事件发现、关键词热度、区域异常 |

方法参考：

- FRED 模式：统一时间序列数据库。
- FRED-MD 模式：多维宏观月度数据标准化。
- KRED 模式：国家级研究数据库 + 主成分提取。
- Nowcasting 模型：用高频替代数据预测低频宏观指标。
- Risk dashboard 模式：用指标阈值和趋势变化生成风险评分。
