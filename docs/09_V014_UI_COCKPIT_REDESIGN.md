# 09｜V0.1.4 首页驾驶舱 UI 改版说明

## 1. 改版目标

V0.1.4 将首页从“功能堆叠型 dashboard”调整为“专业风险驾驶舱”。

核心目标：

1. 顶部第一屏就看到总风险和主要风险来源。
2. 保留圆环仪表盘，提高视觉识别度。
3. 风险模块上移，默认用小圆环展示。
4. 权重预设缩小为芯片，减少占屏。
5. 中国当前 / 历史回溯做成同一套场景切换。
6. 切换到日本 1990 后，下方风险模块同步变化。
7. 数据源控制台下移，并默认折叠。

---

## 2. 新首页结构

```text
1. 顶部标题 + 分析截至日期 + 数据状态
2. 场景驾驶舱
   - 左右切换：中国当前 / 日本1990
   - 总风险圆环
   - 风险等级
   - 历史节点下拉框
3. 关键风险驱动 Top 6
   - 小圆环
   - 模块名称
4. 权重预设芯片
5. 风险模块总览
   - 圆环 / 横条 / 趋势 视图切换
6. 数据源控制台
   - 默认折叠
7. 历史危机相似度
   - 默认折叠
8. 数据源总表
   - 默认折叠
```

---

## 3. 场景切换逻辑

当前支持两个场景：

```text
中国当前状态
日本 1990 资产泡沫破裂 / 资产负债表衰退
```

点击顶部左右箭头切换。

切到日本 1990 后：

- 顶部总分切换为历史节点分数。
- 历史节点下拉框显示在顶部。
- 下方风险模块同步切换为历史节点分数。
- 关键风险驱动 Top 6 同步切换。

---

## 4. 风险模块视图

当前支持：

```text
圆环
横条
趋势
```

说明：

- 圆环：默认首页，适合总览。
- 横条：适合排序比较。
- 趋势：当前为占位，V0.2 接入历史时间序列后显示真实折线。

---

## 5. 数据源控制台

数据源控制台被移到风险模块下方，并默认折叠。

原因：

```text
首页首先应该帮助用户判断风险，而不是先展示配置后台。
```

展开后仍然保留：

- 启用 / 禁用。
- 显示 / 隐藏。
- 是否纳入评分。
- 权重滑块。
- 数据截至日期。
- 来源说明。

## 7. 后续建议

V0.1.5 可以继续做：

1. 风险模块点击展开详情。
2. 圆环视图支持按分数排序 / 按主题排序。
3. 趋势视图接入真实历史序列。
4. 场景增加美国 2008、亚洲 1997、欧债 2009。
5. 数据源控制台增加“仅显示已启用 / 仅显示样例 / 仅显示已验证”。
6. 将企业/店铺/工程项目数据并入主 schema，而不是单独 JSON。
